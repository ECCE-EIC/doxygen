28 // Class description:
29 //
30 // Base Class for realisation of numerical methodes for integration of functions
31 // with signature double f(double) by Gaussian quadrature methods
32 // Roots of ortogonal polynoms and corresponding weights are calculated based on
33 // iteration method (by bisection Newton algorithm). Constant values for initial
34 // approximations were derived from the book: M. Abramowitz, I. Stegun, Handbook
35 // of mathematical functions, DOVER Publications INC, New York 1965 ; chapters 9,
36 // 10, and 22 .
37 //
38 // ---------------------------- Member data: ----------------------------------
39 //
40 // fFunction - pointer to the function to be integrated
41 // fNumber - the number of points in fAbscissa and fWeight arrays
42 // fAbscissa - array of abscissas, where function will be evaluated
43 // fWeight - array of corresponding weights
44 //
45 //
46 // ----------------------------------------------------------------------
47 //
48 // Auxiliary function which returns the value of std::log(gamma-function(x))
49 //
50 // G4double
51 // GammaLogarithm(G4double xx)
