26 // G4PhysicalVolumeStore
27 //
28 // Class description:
29 //
30 // Container for all solids, with functionality derived from
31 // std::vector<T>. The class is a `singleton', in that only
32 // one can exist, and access is provided via the static method
33 // G4PhysicalVolumeStore::GetInstance()
34 //
35 // All solids should be registered with G4PhysicalVolumeStore, and removed on
36 // their destruction. Intended principally for UI browser. The underlying
37 // container initially has a capacity of 100.
38 //
39 // If much additional functionality is added, should consider containment
40 // instead of inheritance for std::vector<T>
41 //
42 // Member data:
43 //
44 // static G4PhysicalVolumeStore*
45 // - Ptr to the single G4PhysicalVolumeStore.
