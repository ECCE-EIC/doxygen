42 // Base class for pre-equilibrium decay models in geant4. By merit of
43 // inheriting from this class a pre-equilibrium decay model can be used
44 // in conjunction with any cascade, string parton model or other high
45 // energy generator in the generation of final states for inelastic scattering.
